## TJH Cinemax Movie Booking System

Overview
A C++ console program for booking cinema tickets. Users register, log in, choose movies, pick seats, pay, and view booking history. Data stores in text files so records stay after closing the program.

Features
* User registration, login, password reset
* Profile view and edit
* Movie listing, filtering, and upcoming shows
* Ticket booking flow with pricing logic
* Seat availability check
* Payment simulation
* Personal booking history from file

Files Used
* Login_names.txt → stores username, password, safe phrase
* Seats_names.txt → stores taken seats
* Booking_confirmation.txt → stores user bookings

Build
Compile using any C++ compiler. I used Embarcadero C++

How to Use
* Start program
* Create account or login
* Select Book Movie or User Profile
* Complete booking and payment
* View history anytime under profile

Good to Know
* Console based
* Simple text storage

Future Enhancements
* Admin control

* Extra booking details
